## Network

The Proxmox server contains two virtual networking environments.

### vmbr0

Production management network.

Purpose

- Internet connectivity
- Management
- Existing services

### vmbr1

Internal enterprise network.

Purpose

- Active Directory
- DHCP
- DNS
- Windows Clients
